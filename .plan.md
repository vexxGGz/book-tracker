# Book Tracker App - Implementation Plan

## Overview
A modern web application for tracking books read throughout the year with analytics, statistics, and real-time price information.

## Tech Stack

### Frontend Framework: React + Vite
- **React 18**: Modern UI library with hooks
- **Vite**: Fast build tool and dev server
- **Tailwind CSS**: Utility-first CSS for clean, modern UI

### Data Storage
- **localStorage**: Client-side data persistence (no backend needed)
- JSON format for easy manipulation

### External APIs
- **Google Books API**: Free with API key (from Google Cloud Console)
  - Fetch book details (title, author, cover, page count)
  - Get current prices via `saleInfo.listPrice` and `saleInfo.retailPrice`
  - Returns price amount and currency code
  - **API Key Setup**: User will need to get free API key from Google Cloud Console
- **Fallback**: Manual price entry if API unavailable

## Data Model

```javascript
Book {
  id: string (UUID),
  title: string,
  author: string,
  isbn: string (for API lookups),
  genre: string,
  pages: number,

  dateAdded: string (ISO date),
  rating: number (1-5 stars),
  review: string (optional),
  price: number,
  currency: string (default: USD),
  coverUrl: string,
  publisher: string (optional)
}
```

## API Integration Details

### Google Books API
- **Endpoint**: `https://www.googleapis.com/books/v1/volumes?q=isbn:{ISBN}&key={API_KEY}`
- **API Key**: Free from Google Cloud Console (users will need to create one)
- **Response includes**:
  - `volumeInfo`: title, authors, pageCount, imageLinks, publisher, categories
  - `saleInfo`: listPrice, retailPrice (with amount and currencyCode)
- **Price Structure**:
  ```json
  "saleInfo": {
    "listPrice": {"amount": 11.99, "currencyCode": "USD"},
    "retailPrice": {"amount": 11.99, "currencyCode": "USD"}
  }
  ```

### Fallback: Open Library API
- **Endpoint**: `https://openlibrary.org/api/books?bibkeys=ISBN:{ISBN}&format=json&jscmd=data`
- Completely free and open
